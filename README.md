# kirchentage.js
### Seite zur Berechnung 
- eines Wochentags anhand des Datums
- Umrechnung von Geburtsdatum, Sterbedatum und Alter
- eines Datums nach Kirchentagen (z.B. 3. Sonntag nach Johannis)
